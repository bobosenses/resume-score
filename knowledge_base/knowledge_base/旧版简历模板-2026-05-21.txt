八、API 接口
============

1. POST /api/search
   请求体：{"query": "...", "top_k": 10, "rerank": true/false}
   返回：
   {
     "query": "...",
     "total_found": N,
     "vector_time_s": 0.02,
     "rerank_time_s": 0.15,
     "vector_results": [...],
     "rerank_results": [...]
   }

2. GET /api/health
   返回：{"status": "ok"}

3. GET /api/collections
   返回：[{"name": "job_functions", "count": 2233}]

4. POST /v1/chat/completions (代理到 vLLM)
   标准 OpenAI API 格式，支持流式和非流式
